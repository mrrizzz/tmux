# TMUX
TERMINAL MULTIPLEXER
## Installation
### Linux
- Make a backup of your current .tmux,conf files: (if it's been existed)
```sh
mv ~/.tmux.conf{,.bak}

```
- Clone the config
```sh
git clone https://github.com/mrrizzz/tmux.git ~/.
cp ~/tmux/.tmux.conf .
```
- Start Tmux!
```sh
tmux
```
